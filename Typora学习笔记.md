# 1. 有序列表

**代码：**

```
1. 萝卜
2. 白菜
```

**显示效果**

1. 萝卜
2. 白菜

**注：**“1.“后面跟了一个空格。

# 2. 无序列表

**代码：**

```

- 白菜
```

**显示效果**

- 萝卜
- 白菜

**注：**“-“后面跟了一个空格。

# 6. 居中可用center标签

**代码：**`<center>我想要居中</center>`

**显示效果**

<center>我想要居中</center>

> 据说Typora目前并不会直接预览居中效果——相应的效果只有输出文本的时候才会显现。
